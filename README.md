# Knights of the Feathered Serpent Productions Website

Static website package for `knightsofthefeatheredserpent.com`.

## Files

- `index.html`
- `about.html`
- `warriors-companion.html`
- `teachings.html`
- `contact.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/images/hero-feathered-serpent.png`
- `assets/images/gallery/` optimized web copies of your reference artwork

## Optional Media

Place the landing-page video at:

```text
assets/videos/journey-to-ixtlan.mp4
```

The homepage already points to that file and falls back to `assets/images/hero-feathered-serpent.png` when the video is not present or cannot autoplay.

## Artwork Replacements

The book cover and chapter plate areas now use the supplied artwork. Future assets can be added under `assets/images/gallery/` and linked from the HTML pages.
